a8k

集成 webpack + react 的最佳实践配置的构建工具

快速上手 →

简洁至上

用最少的配置启动你的React项目开发。

Webpack驱动

享受 React + webpack 的极致开发体验。

高性能

a8k 集成最佳构建实践，你只需关心你的代码，无需关心构建。

# 简单上手

# 安装
yarn global add a8k
# 或者：npm install -g a8k

# 新建一个 项目 文件
k create test

# 进入项目目录
cd test

# 开始开发
k dev

# 构建发布文件
k build

注意

请确保你的 Node.js 版本 >= 8。
